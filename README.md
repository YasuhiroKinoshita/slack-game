# SlackGame

Play games on Slack

This gem provide Controller and Canvas

Improve your like games!

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'slack_game'
```

And then execute:

    $ bundle

## Usage

see `lib/slack_game/game/demo.rb`
